## Get Started

* `npm run build` then open `/dist/index.html`. Or, `npm start` to start the dev server.

## Game Controls

- <kbd>&larr;</kbd> Left
- <kbd>&rarr;</kbd> Right
- <kbd>&uarr;</kbd> Up
- <kbd>&darr;</kbd> Down
- <kbd><em>Space</em></kbd> Pause
- <kbd><em>Enter</em></kbd> Restart
